Developer Documentation

The Developer's Guide contains information related to writing SCIRun modules, Data Types, and other code. Topics such as dataflow programming, memory managment, dynamic compilation, thread managment, data IO are addressed.

The Code Views page is a reference to the organization of SCIRun source code and will help you navigate the source code.

Module references can be located from the Module Reference page. The purpose and use of each module is described by its module reference page.
