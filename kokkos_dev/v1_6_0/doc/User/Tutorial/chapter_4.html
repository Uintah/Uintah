Chapter Overview

In the previous chapter, we constructed three networks to visualize the utahtorso-lowres-voltage dataset. In this chapter, we will merge those networks into one larger network that performs all visualizations concurrently. For details, see the User's Guide 4.15, Navigating a Network. We will add a fourth visualization tool--isosurface extraction.

First, add the following to your canvas:

  • A FieldReader module loading the utahtorso-lowres-voltage.tvd.fld file.

  • Add an Isosurface module from the SCIRun/Visualization menu and connect with the FieldReader.

  • Add a Viewer module and connect to the Isosurface.

Isosurface

The Isosurface takes a volume Field as input and outputs a surface Field or scene graph data (visualized with or without a color map).

Figure 4.1 also shows the Isosurface module UI. The Iso Value slider widget determines the surface value in the incoming volume Field. Upon releasing the slider, the module executes.


Figure 4.1


If a color map is not connected to the Isosurface input port, the module uses the Default Color. By default, the module extracts a surface using the Marching Cubes algorithm, and you can choose the number of threads (processors) the algorithm will use. Or, you may use the NOISE algorithm.

Putting the nets together

Now we will add the visualization techniques from Chapter 3, a procedure that involves sharing the FieldReader and Viewer modules across the 3A, 3B, and 3C networks. You may choose from the following two options for rebuilding the nets you saved in Chapter 3. The first option is to manually reconstruct the nets by,

  • Repeat all of the steps for creating 3A in Chapter 3.

  • Do the same for 3B and 3C, but use the FieldReader and Viewer from 3A.

You can use the complete, color-coded network in Figure 4.2 that shows the previous chapter's networks. Yellow corresponds to 3A, green to 3B, and red to 3C.

Or, as the second option, you may use the File/Insert command to insert all of the nets you saved in Chapter 3, along with their settings. You will need to destroy all but one Viewer module by right-clicking on the module and selecting the Destroy option. To destroy a connection between two modules, right-click on the pipe connecting those modules. In addition to the Viewer, these nets will also share the FieldReader module (in the left uppermost corner of Figure 4.2), which loads the utahtorso-lowres-voltage.tvd.fld file.

You can move module groupings by holding the middle-mouse button and drawing a box that contains the modules you wish to relocate. The selected modules will turn a darker gray. You can then left-click the mouse in order to move the entire group to a new location in the NetEdit frame.

The complete network appears in Figure 4.2. The network is color coded to show the networks from previous chapters, 3A, 3B, and 3C. Yellow corresponds to 3A, green to 3B, and red to 3C. Be sure to save this net since you will use it again in Chapter 6.
