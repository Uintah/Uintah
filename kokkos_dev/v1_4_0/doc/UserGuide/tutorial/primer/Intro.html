Tutorial Introduction

 

Welcome to the SCIRun primer. This tutorial is intended to be an interactive introduction to SCIRun for new users. Since SCIRun is a fairly large system, we will provide a broad overview of SCIRun concepts, and a core set of SCIRun user skills. There are seven chapters in this document. By the end of this document, we expect that the user will have a grasp of dataflow programming, SCIRun architecture, and the specific modules and datatypes that we visit along the way.

Please take your time working through the tutorial, and if you have questions at any time feel free to email the scirun-users@cs.utah.edu list for assistance. You will notice that hyperlinks exist thoughout this document. Each hyperlink points to more comprehensive descriptions about each subject.

Each chapter of tutorial builds on the preceeding one so it is usually best to go through in order. We begin by constructing a very simple, yet functional SCIRun network in Chapter 1, extending it to greater and greater functionality through Chapters 2-7. The first four chapters are only about SCIRun, but starting in Chapter 5, we begin using BioPSE simulation modules to solve a Bioelectric Field Problem. As the tutorial concludes, we take a fresh look at the problem we have incrementally solved in previous chapters.
