BONO

Description:

Template class (Please read!)

Implementation of the Branch-on-Need Octree (BONO) for isosurface extraction in rectilinear scalar fields (see Wilhelms & van Gelder's 1992 ACM ToG paper for details).

Some preprocessing must be done before isosurface extraction (once per data set). Run the module using a preprocessing metafile if this has not already been done.

During execution (run the module on an execution metafile), the user controls isovalue and time value sliders. Data set is read in whenever the time value changes, left in memory when only isovalue changes. A Marching Cubes-type interpolation is performed to create triangles for rendering.

Metafile format:

Preprocessing Execution
PREPROCESS
filename for data file list*
nx ny nz
filename for common tree data
per-tree file base 		EXECUTE
filename for common tree data
filename for per-tree data file list*
miniso maxiso slider_resolution
num_timesteps
* format as follows:
data filename for timestep 0
data filename for timestep 1
data filename for timestep 2
...
 * format as follows:
per-tree file 0 data file (timestep 0)
per-tree file 1 data file (timestep 1)
per-tree file 2 data file (timestep 2)
...

Example metafiles and explanations:

meta_preprocess

PREPROCESS
./filelist
256 256 256
./common_data
./tree
./filelist

/local/sci/raid0/JetData/sclb0000.dat
/local/sci/raid0/JetData/sclb0001.dat
...
/local/sci/raid0/JetData/sclb0099.dat
The preprocess will read ./filelist and create 101 files. ./common_data will contain data common to all trees, and ./tree0 .. ./tree99 will contain time-specific data. The "256 256 256" tells the module that the Jet data set contains 256^3 points. Once the preprocess is finished, the module can be run on:

meta_execute

EXECUTE
./common_data
./trees_file
0 255 0.1
100
./trees_file

./tree0    /local/sci/raid0/JetData/sclb0000.dat
./tree1    /local/sci/raid0/JetData/sclb0001.dat
...
./tree99   /local/sci/raid0/JetData/sclb0099.dat
Since the Jet data set contains byte data (unsigned char), the possible isosurface values range from 0 to 255. This metafile specifies that the isosurface slider should move in increments of 0.1, and that 100 time steps exist.
