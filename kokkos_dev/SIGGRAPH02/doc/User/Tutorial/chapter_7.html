Chapter 7: Feedback

 

ShowDipoles

Chapter Overview

In Chapter 6, we combined simulation and visualization components to form a single, large dataflow network. While the visualization components were dynamic (e.g., you could change the isovalue or streamline seed positions), the models were static. In our final chapter, we will add a feedback module that will allow you to change the position of the dipole model, and interactively visualize the results.

ShowDipoles

We will insert a module into the net from the previous chapter as shown in Figure 7.1. This module is located under the BioPSE/Visualization menu, and is called ShowDipoles. The ShowDipoles module takes in a PointCloud of vectors (the SCIRun representation for a group of dipoles), and allows the user to visualize and interact with those dipoles.


Figure 7.1


ShowDipoles has two outputs: a Field and a Geometry object. The Geometry object attaches to the Viewer module, providing a graphical depiction of the dipoles as arrow widgets.

  1. Select the dipoles by pressing the Shift button and clicking on a dipole with the left mouse button.
  2. Once selected, you can move and rotate the dipoles, depending on which part of the widget you select.
  3. At each release of the mouse button, the latest version of the dipole positions and orientations goes through the Field output port and into the ApplyFEMCurrentSource module.

The ShowDipoles UI (Figure 7.2) has several configuration options for controlling how the dipoles are graphically displayed.

  1. Select the "Normalize Largest" radio button.
  2. Set the Widget Size to 10.
  3. Select "Show Last As Vector."
  4. Unselect "Show Lines."

These options result in a set of dipoles whose lengths have been uniformly scaled, such that the longest vector now has length 1.


Figure 7.2


As the result of connecting the last module, we now have a complete forward finite element network. Figure 7.3 contains the resulting visualization. This is actually the same net that we have provided in the installed SCIRun software as:Packages/BioPSE/nets/forward-fem.net


Figure 7.3


The white arrows in Figure7.3 indicate that you can move the widget around to interactively rotate and shift the torso image (these arrows will not appear in the Viewer when you build this network).

Summary

Please experiment with your completed network. You can explore how different dipole source positions and directions effect the voltage and electric field distributions through the volume.

You can also compare different linear solution methods with the SolveMatrix module, or compare isosurface extraction times using MC (Marching Cubes) versus the NOISE (Near-Optimal IsoSurface Extraction) algorithm. SCIRun is a flexible environment designed for interactive exploration of what-if scenarios.
