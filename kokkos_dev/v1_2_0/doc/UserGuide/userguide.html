SCIRun User Guide

Unix version - Updated May 2, 2001

The SCIRun User's Guide provides a brief introduction to dataflow programming in general and for SCIRun in particular. The basics of using SCIRun are discussed. Additionally, an in depth section is provided for the most frequently used module, the Viewer. Importing and exporting data is also treated.

1 Introduction

The SCIRun User's Guide describes how to use the SCIRun problem solving environment for users who will build and execute networks within the SCIRun environment.

2 Concepts

This section explains the design philosophy and goals of integrated problem solving environments and how BioPSE embodies some of these ideas.

3 Starting SCIRun

This chapter discusses how to start SCIRun.

4 Working with Networks

This section describes how to create, save, open, execute, and edit networks.

5 Visualization with the Viewer

This section describes, perhaps, the most frequently used module of SCIRun, the Viewer, which displays interactive graphical output to the computer screen.

6 Importing Data Into SCIRun

This chapter illustrates some of the channels through which users can import data.

7 SCIRun Modules

Descriptions of all the SCIRun modules, including datatypes used, functions performed and explanations of the user interface elements.
