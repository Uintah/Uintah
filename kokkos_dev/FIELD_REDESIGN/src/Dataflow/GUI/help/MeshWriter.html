MeshWriter Module Help

The MeshWriter module is used for writing meshes in the SCIRun format to disk. It can be used in conjuction with any of the other Mesh modules.

The interface is simple, with a switch to indicate whether the mesh should be saved in binary or ASCII, and then a text window for typing in the name of the file to save to.
