MeshReader Module Help

The MeshReader Module is for reading in meshes that are in the SCIRun format. If the mesh is in the traditional points and tetras files, the files must be run through a conversion program in order to be read in to SCIRun. If there is only a points file, the data can be read in through the PointsReader module and then fed into the Delaunay module to create a mesh.

The interface is a typical filebox interface that allows the user to specify a filter, directory and filename.
