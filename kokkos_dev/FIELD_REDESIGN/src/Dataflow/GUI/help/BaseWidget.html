  • ScaledBoxWidget
  • ScaledFrameWidget
  • ToolWidget
  • ViewWidget
For information on widget related tools see the following:
  • Dialbox
  • MaterialEditor
Shift (or caps lock) must always be depressed to manipulate any widget.

Additionally, all widgets have the following bindings:

  • Left Mouse: Move widget component (drag operation)
  • Middle Mouse: Switch to next mode (click operation)
  • Right Mouse: Pop up widget UI (click operation)
  • Alt Left Mouse: Autoview on widget (click operation)
