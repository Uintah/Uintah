PointsReader Module Help

The PointsReader Module is for reading in a file containing 3D points. The file should start with the number of points and then contain the 3D coordinates X, Y, Z for each point, one point per line.

Example data file:
9
-5.581299 71.269402 -12.054351
-0.976455 75.730783 -52.929733
-2.071106 65.003103 -49.742093
1.782471 98.764571 -35.480275
-3.549246 54.772260 -44.906964
-56.263905 81.456601 -50.025547
-16.341009 105.347639 -12.540367
-51.629313 65.885623 -60.266513
14.604841 75.146363 -18.689893

After the data has been read it, it can be fed into the Delaunay module to create a mesh.

The interface is a typical filebox interface that allows the user to specify a filter, directory and filename.
