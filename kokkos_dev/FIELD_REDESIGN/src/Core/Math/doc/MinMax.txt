
MinMax.h provides simple utility functions for finding the 
minimums and maximums of sets of numbers.

The following functions have been implemented:

// 2 Integers
int Min(int, int);
int Max(int, int);

// 2 doubles
double Min(double, double);
double Max(double, double);

// 3 doubles
double Min(double, double, double);
double Max(double, double, double);
