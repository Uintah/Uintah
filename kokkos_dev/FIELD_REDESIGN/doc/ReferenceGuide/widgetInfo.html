Documentation for Standard tcl or itcl widgets that are used in one of the Problem-Solving Environments
9/27/2000

About this Document

The links that follow are intended to be used to fill our our module documentation. A particular module document can link to information about the widgets used in its graphical user interface. The following links provide man page-type information suitable for developers who want to create a GUI for a new module. scriptics.com currently has man pages for tcl and itcl. They are found at: http://dev.scriptics.com/man/

Please update the list with urls that work. Add a url for a widget once we start using it and remove urls for widgets when we stop using them.

User level documentation is also needed for tcl widgets. User-level documentation details how to interact with the widget to get the result one wants: what mouse button clicks do, what dragging the mouse does, what the widget looks like, etc.
