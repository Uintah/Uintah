Release Notes - SCIRun Version 1.20.3 - Mar 8, 2004

Infrastructure

  • Fixed suspend and foreground bug for RHE3.
  • Fixed unlock/unlock/delete bug in ~Mutex for RHE3.
  • Fixed OSX problem with mutex::lock().
  • Dynamic compilation types are now thread safe.
  • Fixed save image directory listing.
  • Call initialize_ui instead of ui in PowerApps. Before we were calling the ui method and it would not display the GUI window more than once.

Packages

  • Teem: Fixed initialization of TendEvecRGB's evec variable to store integer instead of string.
