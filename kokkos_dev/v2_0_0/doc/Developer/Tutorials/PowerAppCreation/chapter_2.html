Overview

To maintain a uniform look and functionality, PowerApps should be itcl classes and inherit from the SCIRun PowerAppBase class. A PowerApp class should be in a file with a .app extension (i.e. BioTensor.app) to indicate that it is an PowerApp and not a SCIRun network. This file contains the specific itcl class definition and other components discussed below. All .app files should follow the following layout (examples are from BioTensor.app).

  • Output Statement such as "Loading BioTensor (this may take a minute)..."
    puts "\nLoading BioTensor (this may take a
  minute)...\n"
  • References SCIRUN_DATA and SCIRUN_DATASET so the app can run with example data when possible.
