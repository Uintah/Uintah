Release Notes - SCIRun Version 1.22 - July 14, 2004

Content:H1,H2

New Projects in Version 1.22

  • Add support for ImportExport plugins to FieldReader, Field Writer, MatrixReader, MatrixWriter, ColormapReader, and ColormapWriter modules.
  • Removal of the tuple axis in the Teem package. This caused changes in Teem (and sample nets), Insight, DataIO, and Fusion modules.
  • Using Teem 1.7
  • Added --server option to SCIRun. Starts SCIRun with a simple text listener socket that mimics the TCL command prompt.
  • Slaving of View Windows.
  • More environment variables for net saving enhancements.
  • Better Sub-Network support.

Data Sets

  • New "Matlab" directory.
