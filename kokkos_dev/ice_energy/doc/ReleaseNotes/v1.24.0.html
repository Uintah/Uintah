Release Notes - SCIRun Version 1.24 - December 30, 2004

Contents:H1,H2

New Projects in Version 1.24

  • New BioImage PowerApp and tutorial. BioImage is a PowerApp for visualizing regular, three dimensional scalar volumes. BioImage offers 2D visualization of three standard planes: axial, sagittal, and coronal and provides a powerful volume rendering tool that allows the user to interactively visualize an area of interest by assigning a color to a range of values in the volume. It also allows for interactive painting of the Volume Rendering Transfer Function.
  • SCIRun volume visualization dramatically improved. (Now takes advantage of programmable graphics cards.)
  • Added Bundle support. Allows multiple data pipes to be combined into a single pipe. (Can help de-clutter canvas.)
  • Added support for 64 bit linux builds. We have run preliminary tests of SCIRun on 64 bit Altix and Gentoo linux.
  • Added an experimental SCIRUN_NO_PORT_CACHING environment variable. This should allow SCIRun to work on larger data sets.
