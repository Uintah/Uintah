VIEWMESH

Description:

Display an unstructured mesh.

Edges in the mesh are displayed as lines. Some infrastructure has been created for cylinders, but these are effectively unimplemented. Lines look nice and are fast, so cylinders aren't really necessary.

The module takes a .htvol file as input (unstructured rtrt format).
