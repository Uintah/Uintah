Converting yourspec.xml to yourspec.html

The following uses xalan from apache and requires that you have java installed.

your CLASSPATH needs to contain the following:

  • /usr/local/contrib/mcole/java:
  • /usr/local/contrib/mcole/java/bsf.jar:
  • /usr/local/contrib/mcole/java/bsfengines.jar:
  • /usr/local/contrib/mcole/java/xalan.jar:
  • /usr/local/contrib/mcole/java/xerces.jar
    • Your PATH must include your java bin for me it looks like this: /usr/java/bin

    Then do the following:

    cd /usr/local/contrib/mcole/java/Transform
    java Transform yourspec.xml spec.xsl > yourspec.html
    spec.xsl does a very basic translation, nicer ones are sure to follow...
