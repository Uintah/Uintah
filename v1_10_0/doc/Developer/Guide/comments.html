SCIRun Comment Standards

Table of contents

The following list documents the SCIRun Comment Standards. Any questions or requested modifications should be sent to scirun-exec@cs.utah.edu.

  1. Required
  2. Optional
  3. Avoid

Required

  • //! or /*!
  • \author
  • \date
  • \par License
  • - or -# for nested lists

Optional

  • \todo
  • \example
  • \warning
  • \bug
  • \param describe a parameter for a function
  • \image
  • Cross Reference
    • \sa see also:
    • \anchor places an invisble, named anchor into to documentation to which you can refer with the \ref command
    • \ref
    • \link and \endlink
  • Typesetting Commands
    • \em emphesize a word
    • \f$ or \f[ \f] pair: add a math formula in LaTex

Avoid

  • Typesetting Commands
    • \a
    • \b Bold
    • \p
  • LaTex and HTML only commands
    • \latexonly
    • \htmlonly
    • \addindex
  • \skip
  • \line
  • \skipline
  • Conditional Comments
    • \if
  • \section
  • \subsection
