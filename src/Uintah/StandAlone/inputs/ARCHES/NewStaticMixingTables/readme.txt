--------------------------------------------------------------------
	Description of the reaction and mixing files
--------------------------------------------------------------------

1. NG.rxn      - Reaction table for natural gas (Non-adiabatic equilibrium)

2. NG_mix.tbl  - Mixing table for the natural gas used in the flare calculations
		 (Non-adiabatic equilibrium)

3. NG_Beta.mix - Beta mixing table for natural gas created with the NG.rxn (Non-
		 adiabatic equilibrium). Note: Use this table for future validation 
                 calculations. A bug in calculating the equilibrium state is fixed. 
		  

4. JP8_reaction.tbl - Non-adiabatic equilibrium reaction table for JP-8   

--------------------------------------------------------------------
