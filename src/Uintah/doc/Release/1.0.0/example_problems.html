Example Simulations

Overview

The example problems shown here were designed to run on less than about 64 processors and take less than a day to run. Each problem description includes a Problem Description, Simulation Specifics (eg: cell spacing, simulation time, run time, etc), and Results. This information is provided to give new users some idea of what to expect when running similar simulations.
