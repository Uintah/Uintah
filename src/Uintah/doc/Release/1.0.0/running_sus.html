Running a Simulation

SUS Executable Location

The Uintah simulation code is known as sus (which stands for Stand-alone Uintah Simulation). The code has been pre-installed at Lawrence Livermore Nation Laboratory and is accessible at /usr/apps/sus ALC, uP, and Thunder.

Input Files

The sample input files exist in:

/usr/gapps/uintah/src/Packages/Uintah/StandAlone/inputs/UintahRelease/

In this directory are subdirectories for ICE, MPM, ARCHES, IMPM, and MPMICE problems.

In each of these directories are Uintah Problem Specification (.ups) files for each of the sample problems. The command necessary to run Uintah (the sus executable) is found in the example simulation overview writeups.
